# Frontend - Checkout Onboarding App

Frontend desarrollado con React + TypeScript + Vite + Tailwind CSS

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación se ejecutará en `http://localhost:5173`

## Build

```bash
npm run build
```

## Estructura

- `src/pages/` - Páginas principales (Catálogo, Carrito, Checkout)
- `src/components/` - Componentes reutilizables
- `src/services/` - Servicios para comunicación con el API
- `src/context/` - Context API para estado global (Carrito)
